Markdown: Повний курс

Як форматувати документи, не відволікаючись від змісту

Почніть з питання до аудиторії: "Хто з вас вже чув про Markdown або використовував його?" Це допоможе зрозуміти рівень аудиторії.

План лекції

  • Що таке Markdown і чому він популярний?
  • Переваги над MS Word.
  • Базовий синтаксис.
  • Розширені можливості: таблиці, списки завдань.
  • Корисні поради: екранування, лінтери.
  • Магія: формули та діаграми.
  • Екосистема та практичне завдання.

Що таке Markdown?

  • Проста мова розмітки для форматування тексту.
  • Читабельність: документ легко читати навіть без форматування.
  • Конвертація: легко перетворюється в HTML, PDF, DOCX.
  • Філософія: фокус на змісті, а не на зовнішньому вигляді.

Чому він настільки популярний?

  • Простота: Синтаксис інтуїтивно зрозумілий і легко запам'ятовується.
  • Швидкість: Ви форматуєте текст, не відриваючи рук від клавіатури.
  • Портативність: Файли `.md` — це звичайний текст, який можна відкрити будь-де.
  • Сумісність: Ідеально працює з системами контролю версій, як-от Git.

Частина 2: Базовий синтаксис

На цьому етапі можна відкрити текстовий редактор і показувати приклади наживо.

Заголовки: Код


# Заголовок 1 рівня
## Заголовок 2 рівня
### Заголовок 3 рівня

Заголовки: Результат

Заголовок 1 рівня

Заголовок 2 рівня

Заголовок 3 рівня

Виділення тексту: Код


*Курсив* або _Курсив_
**Жирний** або __Жирний__
~~Закреслений~~
***Жирний курсив***

Виділення тексту: Результат

Курсив або Курсив

Жирний або Жирний

Закреслений

Жирний курсив

Списки: Код


- Елемент 1
- Елемент 2
  - Вкладений (відступ)

1. Перший
2. Другий

Списки: Результат

  • Елемент 1
  • Елемент 2
    • Вкладений (відступ)
  1. Перший
  2. Другий

Посилання та зображення: Код


[Google](https://google.com "Підказка")

![placeholder](https://via.placeholder.com/150)

Посилання та зображення: Результат

Google

Цитати та лінії: Код


> Це цитата.
> > Вкладена цитата

---

Цитати та лінії: Результат

Це цитата.
Вкладена цитата

Вставка коду: Код


Рядковий код: `print()`

Блок коду:
```python
def greet(name):
    print(f"Hello, {name}!")
```

Вставка коду: Результат

Рядковий код: print()

Блок коду:


def greet(name):
    print(f"Hello, {name}!")

Частина 4: Корисні поради

Порада 1: Екранування символів

Щоб відобразити спецсимвол, використовуйте зворотний слеш `\`.

Код:


Я хочу написати \*зірочку\*, 
а не курсив.

Результат:

Я хочу написати *зірочку*, а не курсив.

Порада 2: Використовуйте лінтери

  • Markdownlint — інструмент для перевірки стилю.
  • Допомагає писати чистий та консистентний код.
  • Доступний як плагін для VS Code та інших редакторів.

Частина 6: Екосистема та висновки

Де використовується Markdown?

  • Розробка: GitHub, GitLab (README, Issues)
  • Нотатки: Obsidian, Notion, Joplin
  • Статичні сайти: Hugo, Jekyll

Висновок

Markdown — це не просто мова розмітки, а потужний інструмент для підвищення продуктивності.

  • Дозволяє зосередитись на змісті.
  • Легко керувати версіями документів.
  • Працює на будь-якій платформі.
