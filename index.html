Що таке Markdown?

  • Проста мова розмітки для форматування тексту.
  • Читабельність: документ легко читати навіть без форматування.
  • Конвертація: легко перетворюється в HTML, PDF, DOCX.
  • Філософія: фокус на змісті, а не на зовнішньому вигляді.

Частина 2: Базовий синтаксис

Заголовки: Код


# Заголовок 1 рівня
## Заголовок 2 рівня
### Заголовок 3 рівня
#### Заголовок 4 рівня

Заголовки: Результат

Заголовок 1 рівня

Заголовок 2 рівня

Заголовок 3 рівня

Заголовок 4 рівня

Виділення тексту

Код:


*Курсив* або _Курсив_
**Жирний** або __Жирний__
~~Закреслений~~
**_Жирний курсив_**

Результат:

Курсив або Курсив

Жирний або Жирний

Закреслений

Жирний курсив

Списки

Код:


- Елемент 1
- Елемент 2
  - Вкладений

1. Перший
2. Другий

Результат:

  • Елемент 1
  • Елемент 2
    • Вкладений
  1. Перший
  2. Другий

Посилання та зображення

Код:


[Google](https://google.com)

![placeholder](https://via.placeholder.com/150)

Результат:

Google

Цитати та лінії

Код:


> Це цитата.
---

Результат:

Це цитата.

Вставка коду

Код:


```python
def greet(name):
    print(f"Hello, {name}!")

greet("World")
```

Результат:


def greet(name):
    print(f"Hello, {name}!")

greet("World")

Частина 3: Розширені можливості

Таблиці

Код:


| Заголовок 1 | Заголовок 2 |
| :--- | :--- |
| Cell 1 | Cell 2 |
| Cell 3 | Cell 4 |

Результат:

Заголовок 1 Заголовок 2

Списки завдань

- [x] Вивчити базовий синтаксис

  • Вивчити базовий синтаксис

Практичне завдання

Місія: Створити файл PRACTICE.md

Опишіть у ньому свій улюблений фільм або гру, використовуючи:

  1. Заголовок 1-го рівня з назвою.
  2. Короткий опис (цитату) у вигляді блоку цитати.
  3. Жирний та курсивний текст.
  4. Маркований список головних героїв.
  5. Нумерований список 3 причин, чому він вам подобається.
  6. Таблицю з 2 колонок: "Персонаж" та "Актор/Акторка".
  7. Посилання на трейлер на YouTube.
